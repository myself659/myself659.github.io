C/C++方面

  1. 《C与指针》 ☆☆☆☆☆，五星 入门推荐，不要上国产参考书的当
  2. 《 C陷阱与缺陷》 ☆☆☆☆☆，五星 帮你避开常见C语言坑
  3. 《C专家编程》 ☆☆☆☆，四星
  4. 《C++沉思录》 ☆☆☆☆，四星 C++ 进阶之道
  5. 《C++ Prime Plus》 ☆☆☆☆，四星 C++ 入门必备，讲解详细

Python

  1. 《Python 基础教程》☆☆☆☆☆，五星 个人觉得最佳Python入门学习材料，打好基础就用google，github来解决问题

Go

  1. 《Go程序设计语言 英文版》☆☆☆☆，四星

Rust

  1. 《The Rust Programming Language》 ☆☆☆☆，四星 2019

设计模式

  1. 《Head First 设计模式 中文版》

计算机基础

  1. 《深入理解计算机系统 第2版》 ☆☆☆☆☆，五星 请精读且多读几遍，计算机基础方面读这一本书就可以了。
  2. 《计算机程序的构造和解释》 ☆☆☆☆☆，五星 2018

网络编程

  1. 《TCP/IP 详解 卷1：协议》 ☆☆☆☆☆，五星
  2. 《UNIX 网络编程》 ☆☆☆☆，四星
  3. 《Linux 网络编程》 ☆☆☆，三星

  1. 《算法导论》
  2. 《数据结构 C语言版》

Linux

  1. 《鸟哥的Linux私房菜》 ☆☆☆☆，四星 Linux操作入门参考

  3. 《精通Linux驱动程序开发》
  4. 《Linux内核设计艺术》
  5. 《The Linux Programming Interface Handbook》

云计算

  1. 《Software Defined Networks》 ☆☆☆☆，四星 偏架构，需要一定基础与背景
  2. 《腾云 云计算和大数据时代网络技术揭秘》 ☆☆☆☆，四星 通俗易懂地解释网络新技术
  3. 《云计算核心技术剖析》 ☆☆☆，三星
  4. 《云计算与分布式系统》 ☆☆，二星

架构设计

  1. 《SRE Google运维解密》 ☆☆☆☆☆，五星 2017
  2. 《设计数据密集型应用》 ☆☆☆☆☆，五星 2018

数据科学

  1. 《利用Python进行数据分析》 ☆☆☆☆☆，五星 2018

软件工程

安全
