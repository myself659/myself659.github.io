说说写作即学习

September 8, 2023

背景#

写作即学习。

写作是学习体现以下几个方面：

  1. 写作完成学习内容的记录
  2. 写作发现有关学习内容的问题
  3. 写作加深学习内容的记忆与理解
  4. 写作是对学习内容的一种测试方式
  5. 写作促进对学习内容的吸收与发散
  6. 写作引导结学习内容的深入理解与精准描述及表达

如何通过写作来学习#

  1. 完成要点与精华的记录
  2. 记录发现的问题
  3. 通过记录的问题驱动深入思考
  4. 通过写作发现差距并解决gap
  5. 通过写作交付高质量的内容来驱动练习
  6. 在写作的过程中践行费曼技巧，落实图文的方式的解释
  7. 把握写作的触发点：记录，总结，探索，反思，分享，测试对比…

如何保证写作的一致性#

这里一致性是指：

  1. 周期写作如每周写作一篇文章，甚至每天写作一篇
  2. 持续提高写作的质量

迭代写作#

迭代写作有以下好处：

  1. 降低了写作的难度
  2. 有利于长期持续写作
  3. 通过迭代可以一点一点进步，从而不断提高写作的质量
  4. 便于同AI写作结合起来

如何迭代写作呢？

  1. 确定主题与标题
  2. 写框架与要点
  3. 根据框架与要点补充内容
  4. 总结内容
  5. 阅读并完善修改内容
  6. 发布内容
  7. 公众反馈

总结#

通过写作来学习有以下优点：

  1. 有记录的复利
  2. 有迭代的进步
  3. 有交流的反馈
  4. 有一致性的习惯
  5. 有分享的动力与回馈
  6. 有费曼技巧的落实
  7. 有深入思考的问题驱动
