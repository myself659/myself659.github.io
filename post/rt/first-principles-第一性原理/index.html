第一性原理思考101

July 3, 2020

定义#

第一性原理(First Principles)是指从基本原理出发进行思考和解决问题的方法。其具体含义可以概括为:

  1. 不做任何假设,客观看待问题,抛开已有的经验与习惯思维。

  2. 分解问题,回到最基本的真理或事实,以简单的基本要素作为出发点。

  3. 从新角度审视问题的本质,不受已有模式与结论的限制。

  4. 构建由基本原理推导出的自己的原则体系和解决方案。

  5. 持续推敲自己的想法,确保每一步都基于基本原理,而不是来源于别人的观点。

采用第一性原理思考问题,可以消除无效假设,有助于洞察事物的本质,创造出更原创和有效的解决方案。这种思维模式激发我们独立思考,克服思维定势,是许多优秀创新者如伊隆·马斯克所推崇的一种高效思考方法。

简而言之,第一性原理就是保持开放和怀疑的态度,不依赖其他任何假设和理论,直接从基本原理出发建立理论体系的方法。它追求从最基本的层面来理解和描述物质世界的本质规律。这有助于我们跳出框架,从新的视角看待世界。

如何得到最基本的事实与原理#

如何得到最基本的事实与原理呢？这里推荐两个方法

  1. 苏格拉底式提问法
  2. 五个为什么提问法

苏格拉底式提问法#

苏格拉底式提问法同普通讨论之间的关键区别在于，前者试图以系统的方式得出第一性原理。苏格拉底式提问法通常遵循以下流程。

  1. 澄清你的想法并解释想法的来源。（为什么我会这么想？我到底是怎么想的？）
  2. 对假设提出挑战。（我怎么知道这是真的？如果我的想法正好相反呢？）
  3. 寻找论据。（我该如何支撑自己的论点？论据来源于哪里？）
  4. 思考其他可能的视角。（别人可能有什么想法？我怎么知道自己是对的？）
  5. 探究后果和影响。（如果是我错了，该怎么办？如果我错了，会有什么后果？）
  6. 质疑最初的问题。（我为什么会那样想？我的想法对吗？从推理过程中，我能得出什么结论？）

五个为什么提问法#

五个为什么提问法是一种通过连续提问"为什么"来深入探究问题根源的方法。它可以帮助我们追根溯源,深入思考并找到真正的原因。 这个方法主要遵循以下几个步骤:

  1. 明确问题:首先要确认具体要探讨的问题是什么。
  2. 问第一个为什么:对这个问题提出第一个为什么,试图找到并表达出现这个问题的第一个原因。
  3. 继续发问为什么:针对第一个原因再次发问为什么,探索更深层次的原因,类推下去。
  4. 重复提问:每找到一个原因,就继续追问为什么,通常需要重复提问5次。
  5. 找到根本原因:通过不断发问为什么,可以剥离表层,深入探究,找到问题的根本原因。
  6. 提出解决方案:当找到真正的根本原因后,就可以针对性地提出预防或解决问题的方法。

采用这个简单的提问方式,可以帮助我们逻辑性地分析探讨问题,不断深入,找到真正的症结所在。这是一个可以洞悉问题本质与第一性的有效方法。

过程#

从学习思考过程看，第一性原理思考有两个阶段：

  1. 从问题与现象到根本原因与本质的自上而下的思考
  2. 从确定的第一性原理到解决方案的自下而上的思考

整个第一性原理思考的过程如下：

  1. 描述问题（假设、目标）并明确问题（假设、目标）
  2. 将问题（假设、目标）分解子类
  3. 确定问题（假设、目标）的第一性基本原理
  4. 根据第一性基本原理从头构建新的解决方案

难点#

  1. 大脑的思考会消费大量的能量，大部分情况下大脑的思考路径是：问题-方案，而第一性原理思考的路径是：问题-第一性原理-方案，大脑往往会不适应第一性原理思考的过程。

  2. 从问题到第一性原理往往是一个很长的链路过程，整个链路是一环套一环，要确保整个过程是一件不容易的事情

  3. 第一性原理思考要求深度思考

  4. 第一性原理思考要求有怀疑一切，大胆假设，小心求证

  5. 第一性原理思考要求think out of box

本质 vs 第一性原理#

本质指事物存在和发展的最基本、最核心的属性或规律。本质是事物内在存在的客观实体,揭示本质是为了认识事物的真正特征。 第一性原理是指探究一个系统的最基本规律和最核心要素。第一性原理思考强调主体的能动作用,通过抽离、简化,重新构建对系统的认知框架。

两者主要区别有:

  1. 本质更强调被探究对象的客观性,是存在的、不可改变的。第一性原理更强调主体的主观作用,是可以重新构建的。
  2. 本质需要发现,而第一性原理需要创造。揭示本质是对真理的发现,而建立第一性原理需要主体的创造性思维。
  3. 本质具有内在性,属于事物自身的属性。第一性原理具有构造性,是主体在一定条件下构建的。
  4. 本质更多是分析性的,而第一性原理更多关注如何综合元素重新建构。
  5. 本质强调“是什么”,而第一性原理强调“能怎样”。 总之,本质是对客观存在的描述,而第一性原理是主观上的新概念化和简化。但两者都希望达到揭示事物深层特征和规律的目的

注意#

  1. 不要离开系统谈第一性原理
  2. 第一性原理主要用于理解本质，创新，优化，决策，体系结构等场景
