前言

作为一个常年以c,c++,golang为主的后端程序员，刚接触nodejs有一个困惑，就是node有哪么多版本到底该选择哪个版本呢？

node版本

根据需求选择版本，那么node版本有哪些特点呢？

  1. 从node 6 开始支持ES6
  2. 从node 8 开始支持Async Await
  3. node 10 增加了promisified fs模块
  4. node 12增加了ES6 modules

从上面应该注意到node偶数版本是稳定版本（类似于linux）。
