Telsa买入bitcoin意味着什么

February 10, 2021

background#

最近Telsa投入15亿美元买入了bitcoin，消息一出，bitcoin应声大涨，这事激起广泛的关注。

telsa买入bitcoin的原因#

为什么telsa买入比特币？个人认为有以下原因：

  1. 在全球继续放水的预期下，telsa公司资产配置的需要
  2. telsa买入bitcoin，可以赢得加密货币富豪的支持，这些人有很强的消费能力
  3. 传统金融政策难以持续，买入bitcoin可以试水新的货币机制，也可以赢得先机
  4. bitcoin时隔3年重回2万美元，并继续突破了3万美元，4万美元等历史新高，对于比特币来说，它的反脆弱性得到证明
  5. bitcoin得到paypal与square等公司支付业务的支持

为什么只买入bitcoin而没有买入其他的加密货币如ethereum呢？个人认为有以下原因：

  1. bitcoin市值最高，占比超过了整个加密货币的60%
  2. bitcoin相比较于其他加密货币风险更小
  3. bitcoin共识长达10多年，共识稳固
  4. bitcoin具有抗通胀的属性，而ethereum现在每年都有一定的通胀
  5. 总之，bitcoin是加密货币的始姐，也是最强的头部

意义与影响#

telsa买入比特币意义与影响有哪些？

  1. bitcoin得到世界顶级精英与公司的认可，也意味加密货币得到认可，为整个加密货币市场注入活力，吸引了用户，资产及人才。
  2. 意味牛市继续，很有可能是长牛
  3. 为其他机构与公司提供了入场的参考，telsa进场了，apple，amazon，google，microsoft下一步也会来吗？
  4. Elon Musk及其telsa的影响力有利于加密货币与区块链技术的普及
