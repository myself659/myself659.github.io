做一名通才吧

October 15, 2019

背景#

这是一个信息爆炸，连接无限，复杂快变的时代。

通才#

通才(Polymath),指的是那些具有广泛涉猎知识和兴趣,在不同领域都有深入理解与创造力的人。其主要特点如下：

  1. 广阔的视野
  2. 灵活的思维
  3. 强烈的好奇心和广泛的兴趣
  4. 心智富有弹性，有深度也有广度
  5. 跨领域切换与实践

通才的优点#

成为一名通才具有以下优点：

  1. 通才可以将技能库的任意两项或多项技能组合出一项新的技能

  1. 通才有丰富的知识与技能储备，有利于快速学习新的技能

  2. 通才更容易玩跨界

  3. 通才更容易适应未来的变化

  4. 通才思想开放，思维活跃，不局限某一个领域有利于解决复杂问题

  5. 通才在市场上（不仅是就业市场）有更多的机会

  6. 通才具有更强的反脆弱性

  7. 通才有很强的组合能力与创新能力（大多数创新来自于组合）

  8. 通才利于领导与管理

如何成为通才呢？#

  1. 学习常用的心智模型，心智模型是通用的，是跨学科的

  2. 从基础开始

  3. 学习如何学习，如何快速学习，如何构建自己的知识体系

  4. 练习抽象总结及举一反三

  5. 多角度研究最典型的案例

  6. 兴趣广泛，广泛阅读，培养好奇心

  7. 早动手，多实验，多实践

  8. 项目驱动学习
