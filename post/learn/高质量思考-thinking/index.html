什么是高质量的思考

June 3, 2017

思考的质量#

人一思考,上帝就开始发笑。所以人要学会思考，提高思考的质量。

下面从四个角度来看看有哪些因素决定思考的质量？

原则#

  1. 怀疑一切，大胆假设，小心验证
  2. 实事求是
  3. 注重方法与策略
  4. 主动积极

Input#

  1. 好问题
  2. 问题描述
  3. 能量
  4. 专注

Process#

  1. 广度
  2. 深度
  3. 清晰
  4. 适度
  5. 创造

Output#

  1. 清晰
  2. 准确
  3. 全面
  4. 深度
  5. 逻辑
  6. 创造
  7. 可操作性
  8. 持续性

总结#

高质量思考= 高质量的原则+高质量的Input+高质量的Process+高质量的Output。
