说说学到位

April 18, 2020

什么是学到位#

学到位可以分解为四个到位：

  1. 理解到位
  2. 体系到位
  3. 融合到位
  4. 应用到位

理解到位#

怎么做到理解到位？

不多说，推荐费曼学习法。

If you can’t explain it simply, you don’t understand it well enough. 如果你不能简单明了地解释一件事情，那么说明你还没有很好地理解它。

体系到位#

应用第一性原理进行学习。

参考elon musk的方法：将知识视为一颗树，首先确保你理解基本原则，也就是树干与主要树枝，然后再深入细节，即树叶。

基础先行。越基础越重要，基础牢固，知识体系也才会牢固。

融合到位#

了解知识，理解知识，这些都是为了实践中应用知识，让知识为自己服务。

应用知识是一个复杂的活动，应用知识往往并不是一个简单知识点的应用，而是一系列知识点的应用。

所以应用知识之前，我们需要融合知识，实现举一反三，触类旁通。

如何融合知识呢？

那就是新学习的知识与原来的知识体系联系起来，通过尽可能地方式将知识联系起来，串联在一起，融合在一起。

思维导图就是一种很好的连接与融合的形式。

应用到位#

将知识应用到具体一个问题，解决一个问题，甚至解决一类问题。从理论走向实践，知行结合，真正地让知识为自己服务。

总结#

理解是基础,体系是延伸,融合是内化,应用是升华。
