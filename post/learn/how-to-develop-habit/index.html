如何养成好习惯

December 6, 2019

背景#

养成好习惯是一项复杂的任务，但通过科学的方法，我们可以使其变得更加简单和高效。

以下将从"原子习惯"、“微习惯"和"Hook模式"三个策略，以及实践原则，来探讨如何养成好习惯。

原子习惯#

根据James Clear的《原子习惯》一书，我们可以通过四个步骤来建立一个习惯循环（Habit Loop）：

1.提示（Cue） 2.渴望（Craving） 3.反应（Response） 4.奖励（Reward）。

对应这四个步骤，Clear也给出了四个原则：

  1. Make it obvious：让习惯的提示尽可能明显。例如，如果你想养成每天刷LeetCode的习惯，你可以将LeetCode的图标放在你电脑桌面最显眼的位置。

  2. Make it attractive：让习惯本身具有吸引力。你可以把刷LeetCode看作是一种挑战，每解决一个问题，就是你技术能力的提升。

  3. Make it easy：让习惯的执行尽可能简单。你可以从解决简单的问题开始，让自己逐渐习惯于这个过程。

  4. Make it satisfying：让习惯的奖励尽可能满足。例如，每解决一个LeetCode的问题，你就给自己一些小奖励，如休息一会儿或者看一集美剧。

微习惯#

微习惯是指那些每天至少会做一次、完成不需要费力费心、并且完成时间不超过2分钟的习惯。微习惯的触发与完成一般是这样的：一旦某事发生，我就会完成这个习惯。例如，一旦我起床了，我就会立即喝一杯水。这种微小的行为可以帮助我们养成更大的习惯，因为它们可以作为一个习惯的初始阶段，让我们更容易开始。

Hook模式#

根据Nir Eyal的《Hooked: How to Build Habit-Forming Products》一书，习惯的形成也可以通过四个步骤：触发器（Trigger）、行动（Action）、奖励（Reward）和投入（Investment）。这个模式强调了投入的重要性，即我们在习惯中投入的越多，就越可能坚持这个习惯。

实践原则#

在实际养成习惯的过程中，我们还需要遵循一些原则：

  1. 从最简单最容易成功的习惯开始：简单的习惯更容易坚持，而且能让你建立起成功的信心。

  2. 一次只专注于培养一个习惯：尽管我们可能有很多想要改变的习惯，但是一次只改变一个习惯可以让我们更专注，提高成功的几率。

  3. 事不过三：这是一个防止习惯断链的策略。即使我们有时候无法坚持习惯，也不要连续两天错过一个习惯，以避免陷入恶性循环。

通过以上的原子习惯、微习惯和Hook模式，以及遵循实践原则，我们可以更科学、有效地养成好习惯，从而提高生活的质量和工作效率。

总结#

养成好习惯并不是一蹴而就的事情，它需要时间、耐心和恒心。然而，通过使用《原子习惯》、《微习惯》、《Hooked: How to Build Habit-Forming Products》中的策略，我们可以更快、更有效地建立起想要的习惯。

让我们以行动说话，从最简单的微习惯开始，逐步培养和强化习惯，最终实现自我提升和成长。
