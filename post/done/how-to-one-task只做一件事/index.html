只做一件事，避免多任务并发

June 3, 2019

多任务并发的危害#

  1. 人类大脑结构上并不适合多任务处理,强行进行多任务会降低效率与产出
  2. 多任务带来的切换成本开销大
  3. 多任务处理会降低工作质量，增大压力
  4. 多任务加大大脑的负担，容易出错
  5. 频繁进行任务切换会形成反馈循环,获得即时满足感但实际效果很少，如果长时间地进行这种循环，大脑很难建立延迟循环
  6. 多任务降低深度工作的可能性，导致工作做不到位
  7. 多任务影响智商，可能伤害大脑

只做一件事#

应对多任务并发的最好的办法就是一个时间块只做一件事。这样做有如下好处：

  1. 采用时间块管理提高时间管理的效率
  2. 分解任务，提高任务的可执行性
  3. 任务优先级管理
  4. 单位时间块目标单一明确
  5. 有利于投入与保持专注，便于进入心流状态，提高效率与产出
  6. 有利于将工作做到位

如何做到在一个时间块只做一件事呢？#

  1. 避免时不时检查邮箱，通知等不良习惯
  2. 使用onetab，避免多tab环境（多tab容易引起多任务与注意分散）
  3. 使用专注模式
  4. 如果任务不需要使用互联网，主动block互联网
  5. 有意识只做一件事并进行长时间的训练
  6. 主动使用一个时间块的背景音乐引导专注一件事
